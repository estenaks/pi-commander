Random filters
Color identity (exactly):
“Exactly” means the random commander's color identity must be equal to the checked set (e.g. only {R} or exactly {U,B}, etc). If nothing is checked, random is any color identity.
Mana value (cmc)
Enter an exact mana value (3), an operator (>=3), or a range (2..4). This value is used by Momir Vig and may also be sent with the commander request.
Card type filters (inclusive)
Checked types are added with AND semantics (t:type t:other). If you want OR semantics (e.g. creature OR artifact) you can later extend the UI to send raw q.
Arbitrary is: labels
Use this to pass arbitrary Scryfall is: labels (e.g. is:commander, is:funny). Multiple comma-separated entries are supported.
